### Критические ошибки

**Отсутствие аргументов CLI** Запуск без аргументов командной строки
**Отсутствие флага -a** Не указан IP-адрес 
**Отсутствие флага -p** Не указан порт
**Отсутствие флага -r** Не указана роль 
**Отсутствие флага -i** Не указан индекс 
**Отсутствие флага -L** Не указана библиотека 
**Некорректный порт** Порт ≤ 0 или не число
**Некорректный индекс** Индекс не число
**Неизвестный флаг** Передан несуществующий флаг
**Превышение лимита команд** >5 неизвестных команд подряд
**Ошибка чтения** Невозможно прочитать команду

### Некритические ошибки

**Некорректный ввод числа** Ввод букв вместо цифр 
**Непечатные символы** Символы \\t, \\n, \\r в команде 
**Пустое имя программы** Ввод пустой строки в name 
**Неверный тип вектора** Ввод не int/float/double 
**Нет типа вектора** Ввод vector без выбора типа 
**Неизвестная команда** Ввод несуществующей команды 
**Нулевая w-компонента** Последний элемент вектора = 0
